# Foodstack

A card/cooking puzzle game built with Phaser 3 and TypeScript.

## Tech Stack

- **Language:** TypeScript
- **Game Engine:** Phaser 3
- **Bundler:** Vite
- **Linter:** oxlint
- **Formatter:** oxfmt
- **Package Manager:** pnpm

## Commands

| Command             | Description                      |
| ------------------- | -------------------------------- |
| `pnpm dev`          | Start dev server                 |
| `pnpm build`        | Production build                 |
| `pnpm preview`      | Preview production build         |
| `pnpm typecheck`    | Run TypeScript type checking     |
| `pnpm lint`         | Lint `src/` with oxlint          |
| `pnpm format`       | Format `src/**/*.ts` with oxfmt  |
| `pnpm format:check` | Check formatting without writing |

## Skills

- When working on Phaser-related tasks (scenes, sprites, physics, tilemaps, animations, input, game architecture), always use the `/phaser-gamedev` skill.
